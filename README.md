# Калькулятор жорсткості поліуретанових вставок

Інструмент для порівняння жорсткості на згин тонких поліуретанових пластин залежно від товщини, ширини та твердості матеріалу за шкалою Shore A.

Розроблено для підбору оптимальних параметрів вставок у гімнастичних наплічниках.

---

## Як користуватись

1. Введіть **ширину** смужки `b` (мм) — спільна для всіх варіантів
2. Для кожного варіанту задайте **товщину** `t` (мм) та **твердість** Shore A
3. Калькулятор миттєво показує модуль Юнга `E`, жорсткість `EI` та порівняння між варіантами

---

## Розрахунки

### 1. Shore A → Модуль Юнга (E)

Твердість за Shore A не є прямою мірою жорсткості — вона вимірює опір поверхні вдавлюванню. Для розрахунків потрібен **модуль Юнга E** (МПа), який показує, наскільки матеріал опирається деформації під навантаженням.

Використовується апроксимаційна формула для еластомерів:

```
E (МПа) = 0.0981 × (56 + 7.66 × Shore A)
```

| Shore A | E (МПа) |
| ------- | ------- |
| 70      | 52.6    |
| 80      | 65.5    |
| 85      | 69.3    |
| 90      | 73.2    |
| 95      | 77.1    |

> **Примітка:** формула є апроксимацією і дає точність ±15–20% залежно від конкретного матеріалу. Для точних розрахунків рекомендується використовувати дані виробника поліуретану.

---

### 2. Жорсткість на згин (EI)

Жорсткість пластини на згин розраховується за формулою балки прямокутного перерізу:

```
EI = E × b × t³ / 12
```

де:

- `E` — модуль Юнга (МПа)
- `b` — ширина смужки (мм)
- `t` — товщина пластини (мм)
- `EI` — жорсткість на згин (МПа·мм⁴)

**Ключовий висновок:** жорсткість пропорційна **кубу товщини** (`t³`). Це означає, що зміна товщини має набагато більший вплив, ніж зміна твердості:

| Зміна                           | Вплив на EI |
| ------------------------------- | ----------- |
| 85 → 90 Shore A (товщина 2 мм)  | +6%         |
| 2 мм → 2.5 мм (85 Shore A)      | +95%        |
| 2 мм → 1 мм (будь-який Shore A) | −87.5%      |

---

### 3. Порівняння варіантів

Калькулятор показує:

- **Горизонтальні бари** — відносна жорсткість варіантів (100% = найжорсткіший)
- **Коефіцієнти** — у скільки разів варіант жорсткіший або м'якший за перший варіант

---

## Обмеження моделі

- Формула Shore A → E є **апроксимацією** (Gent, 1958), придатною для швидкого порівняння, але не для точного інженерного розрахунку
- Модель розглядає пластину як **просту балку** — не враховує защемлення країв, залишкові деформації та нелінійну пружність еластомерів
- При товщині менше 2 мм вимірювання Shore A на реальному зразку дає завищені показники (стандарт ASTM D2240 вимагає мінімум 6 мм)
- Результати є **відносними** — використовуйте їх для порівняння варіантів між собою, а не як абсолютні значення жорсткості
